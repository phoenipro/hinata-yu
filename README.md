# 陽詩夕 Web Site

VTuber「陽詩夕(ひなたゆう)」さんの公式リンクとローカル画像素材をまとめた静的サイトです。

## メモ

- 2026年6月17日に、ユーザー提供の公式URL群を確認しました。
- YouTube の表示名は「陽詩夕 HinataYu」、FANBOX のページタイトルは「陽詩夕(ひなたゆう)」です。
- YouTube RSSと動画概要欄から、音楽制作、MIX、DTM講座、ゲーム実況、雑談ラジオ、セルフネガティブ自己紹介といった活動要素を読み取り、サイト本文に反映しています。
- `_img` フォルダ内のキャラクター画像から色を抽出し、サイトカラーに反映しています。

## 開き方

`index.html` をブラウザで開いてください。
